Create a matrix descriptor for multiple matrices (an MPS::Matrix object can store multiple matrices).

Parameters

  • rows – Number of rows

  • columns – Number of columns

  • matrices – Number of matrices

  • rowBytes – Number of bytes in each row. Given by (number of columns * size of data type)

  • matrixBytes – Number of bytes in each matrix.

  • dataType – Data type of the elements being stored.
